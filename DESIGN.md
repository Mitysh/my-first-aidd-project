# Design Doc — Issue #1: Веб-страница с кнопкой «Счастье»

## Ссылка на задачу

https://github.com/Mitysh/my-first-aidd-project/issues/1

## Описание

Создать простую одностраничную веб-страницу с кнопкой, которая при нажатии показывает всплывающее сообщение о счастье.

## Требования

| # | Требование | Приоритет |
|---|-----------|-----------|
| 1 | Файл `index.html` | Обязательно |
| 2 | Большая кнопка по центру страницы | Обязательно |
| 3 | Текст кнопки: «Нажми, чтобы получить счастье» | Обязательно |
| 4 | При клике — `alert("Вы абсолютно счастливы! 🚀")` | Обязательно |
| 5 | Минималистичный, приятный дизайн | Обязательно |

## Технические решения

- Один файл `index.html` со встроенными стилями (`<style>`) и скриптом (`<script>`). Внешние зависимости не нужны.
- CSS Flexbox для центрирования кнопки по вертикали и горизонтали.
- Мягкий градиентный фон, скруглённая кнопка с hover-эффектом.
- Семантический HTML, `lang="ru"`, корректная кодировка UTF-8.

## Структура файлов

```
index.html   — единственный файл, содержит HTML + CSS + JS
DESIGN.md    — этот документ
README.md    — описание проекта (уже существует)
```

## Дополнения (по ходу работы)

- Вместо одного фиксированного alert добавлен генератор случайных фраз (12 вариантов). При каждом нажатии — новая позитивная фраза.

## Критерии готовности

- Страница открывается в браузере без ошибок.
- Кнопка визуально по центру экрана.
- Клик по кнопке вызывает alert со случайной позитивной фразой.
- Дизайн чистый и минималистичный.
